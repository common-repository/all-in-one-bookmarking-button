== Description ==
It adds a SocialMarker button at the bottom of every post which helps add a post to all major social bookmarking services and news sites in a few minutes. All submission data is auto-completed.

== Installation ==

   1. Download the Plugin.
   2. Upload the contents of the zip file to wp-content/plugins/
   3. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==

= What are the requirements for this plugin? =

WordPress 4.0 or above 

= What services will my post be added to? =

The user has the ability to decide which and how many of the services he will use to socially bookmark your post.

== A brief Markdown Example ==

* one button to cover all social bookmarking and news sites
* fields for submission are auto-completed
